# Mood Audio Assets

Place audio files here for each mood's ambient music/sounds.

## Required files (MP3 format, ~2-5MB each, loop-friendly):
- mood_happy.mp3     — Birdsong + gentle acoustic guitar
- mood_sad.mp3       — Soft piano + slow rain + cello
- mood_anxious.mp3   — Forest ambience + theta binaural beats (4-8Hz)
- mood_angry.mp3     — Ocean waves at ~6 breaths/min rhythm
- mood_confused.mp3  — Soft Bach/Mozart orchestral

## Free sources:
- https://freemusicarchive.org
- https://pixabay.com/music (search: "rain", "ocean", "forest", "birdsong")
- https://mynoise.net (binaural beats, nature sounds)
- https://musopen.org (classical music, royalty-free)

## Tips:
- Use loop-friendly files (start and end at similar volume)
- Keep files under 5MB for fast loading
- Upload to Firebase Storage: /mood_assets/audio/
